Project Hello Quad

DigiPen Graphics HW

The Hello Quad project was a part of my CS250 Computer Graphics course at DigiPen. The goal of this assignment was to get an OpenGL project building & running for Windows & Web by displaying a simple animated quad.

Tasks Completed

  • Implemented classes to encapsulate OpenGL Textures, VertexBuffers, IndexBuffers, and VertexArrays.
  • Created a pair of vertex and fragment shaders to display a quad mesh.
  • Updating the Demo logic to feed the animations based on time, frame count, and mouse position.
  • Create my own animations in the shader code to display the quad in a fun way.

Reflection

During development, I identified an overlooked issue where the shader animations were not synchronizing correctly with the frame updates. By implementing a time-based uniform in the shaders, I was able to achieve smooth and synchronized animations. This project significantly enhanced my understanding of shader programming and creating a starting point for OpenGL graphics development.
